##市场活动


###项目介绍
提供运营活动的服务，包括运营位、卡券、兑换码

1. 为前端提供http服务，运营位展示、用户卡券列表、活动领取红包、兑换码使用
2. 为lender提供http卡券使用服务，包括卡券使用、锁定、使用回调、对账
3. 为jrockt2提供dubbo服务
